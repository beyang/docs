NAME
       docker-container-rename - Rename a container

SYNOPSIS
       docker container rename CONTAINER NEW_NAME

DESCRIPTION
       Rename a container.  Container may be running, paused or stopped.

OPTIONS
       -h, --help[=false]
           help for rename

SEE ALSO
       docker-container(1)

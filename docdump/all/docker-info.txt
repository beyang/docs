NAME
       docker-info - Display system-wide information

SYNOPSIS
       docker info [OPTIONS]

DESCRIPTION
       Alias for docker system info.

OPTIONS
       -f, --format=""
           Format the output using the given Go template

       -h, --help[=false]
           help for info

SEE ALSO
       docker(1)

NAME
       docker-plugin-disable - Disable a plugin

SYNOPSIS
       docker plugin disable [OPTIONS] PLUGIN

DESCRIPTION
       Disable a plugin

OPTIONS
       -f, --force[=false]
           Force the disable of an active plugin

       -h, --help[=false]
           help for disable

SEE ALSO
       docker-plugin(1)

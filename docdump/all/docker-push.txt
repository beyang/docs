NAME
       docker-push - Push an image or a repository to a registry

SYNOPSIS
       docker push [OPTIONS] NAME[:TAG]

DESCRIPTION
       Alias for docker image push.

OPTIONS
       --disable-content-trust[=true]
           Skip image signing

       -h, --help[=false]
           help for push

SEE ALSO
       docker(1)

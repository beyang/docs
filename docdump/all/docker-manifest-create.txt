NAME
       docker-manifest-create - Create a local manifest list for annotating
       and pushing to a registry

SYNOPSIS
       docker manifest create MANIFEST_LIST MANIFEST [MANIFEST...]

DESCRIPTION
       Create a local manifest list for annotating and pushing to a registry

OPTIONS
       -a, --amend[=false]
           Amend an existing manifest list

       -h, --help[=false]
           help for create

       --insecure[=false]
           Allow communication with an insecure registry

SEE ALSO
       docker-manifest(1)

NAME
       docker-service-ls - List services

SYNOPSIS
       docker service ls [OPTIONS]

DESCRIPTION
       List services

OPTIONS
       -f, --filter=
           Filter output based on conditions provided

       --format=""
           Pretty-print services using a Go template

       -h, --help[=false]
           help for ls

       -q, --quiet[=false]
           Only display IDs

SEE ALSO
       docker-service(1)

NAME
       docker-stack - Manage Docker stacks

SYNOPSIS
       docker stack [OPTIONS]

DESCRIPTION
       Manage Docker stacks

OPTIONS
       -h, --help[=false]
           help for stack

       --kubeconfig=""
           Kubernetes config file

       --orchestrator=""
           Orchestrator to use (swarm|kubernetes|all)

SEE ALSO
       docker(1), docker-stack-deploy(1), docker-stack-ls(1),
       docker-stack-ps(1), docker-stack-rm(1), docker-stack-services(1)

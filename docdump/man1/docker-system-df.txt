NAME
       docker-system-df - Show docker disk usage

SYNOPSIS
       docker system df [OPTIONS]

DESCRIPTION
       Show docker disk usage

OPTIONS
       --format=""
           Pretty-print images using a Go template

       -h, --help[=false]
           help for df

       -v, --verbose[=false]
           Show detailed information on space usage

SEE ALSO
       docker-system(1)

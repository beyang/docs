NAME
       docker-wait - Block until one or more containers stop, then print
       their exit codes

SYNOPSIS
       docker wait CONTAINER [CONTAINER...]

DESCRIPTION
       Alias for docker container wait.

OPTIONS
       -h, --help[=false]
           help for wait

SEE ALSO
       docker(1)

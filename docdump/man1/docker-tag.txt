NAME
       docker-tag - Create a tag TARGET_IMAGE that refers to SOURCE_IMAGE

SYNOPSIS
       docker tag SOURCE_IMAGE[:TAG] TARGET_IMAGE[:TAG]

DESCRIPTION
       Alias for docker image tag.

OPTIONS
       -h, --help[=false]
           help for tag

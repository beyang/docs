NAME
       docker-plugin-rm - Remove one or more plugins

SYNOPSIS
       docker plugin rm [OPTIONS] PLUGIN [PLUGIN...]

DESCRIPTION
       Remove one or more plugins

OPTIONS
       -f, --force[=false]
           Force the removal of an active plugin

       -h, --help[=false]
           help for rm

SEE ALSO
       docker-plugin(1)

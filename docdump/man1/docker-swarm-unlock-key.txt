NAME
       docker-swarm-unlock-key - Manage the unlock key

SYNOPSIS

       Manage the unlock key

OPTIONS

           help for unlock-key

       -q, --quiet[=false]
           Only display token

       --rotate[=false]
           Rotate unlock key

NAME
       docker-context-rm - Remove one or more contexts

SYNOPSIS
       docker context rm CONTEXT [CONTEXT...]

DESCRIPTION
       Remove one or more contexts

OPTIONS
       -f, --force[=false]
           Force the removal of a context in use

       -h, --help[=false]
           help for rm

SEE ALSO
       docker-context(1)

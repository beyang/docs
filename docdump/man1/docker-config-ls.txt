NAME
       docker-config-ls - List configs

SYNOPSIS
       docker config ls [OPTIONS]

DESCRIPTION
       List configs

OPTIONS
       -f, --filter=
           Filter output based on conditions provided

       --format=""
           Pretty-print configs using a Go template

       -h, --help[=false]
           help for ls

       -q, --quiet[=false]
           Only display IDs

SEE ALSO
       docker-config(1)

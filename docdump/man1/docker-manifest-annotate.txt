NAME
       docker-manifest-annotate - Add additional information to a local image
       manifest

SYNOPSIS
       docker manifest annotate [OPTIONS] MANIFEST_LIST MANIFEST

DESCRIPTION
       Add additional information to a local image manifest

OPTIONS
       --arch=""
           Set architecture

       -h, --help[=false]
           help for annotate

       --os=""
           Set operating system

       --os-features=[]
           Set operating system feature

       --variant=""
           Set architecture variant

SEE ALSO
       docker-manifest(1)

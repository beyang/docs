NAME
       docker-save - Save one or more images to a tar archive (streamed to
       STDOUT by default)

SYNOPSIS
       docker save [OPTIONS] IMAGE [IMAGE...]

DESCRIPTION
       Alias for docker image save.

OPTIONS
       -h, --help[=false]
           help for save

       -o, --output=""
           Write to a file, instead of STDOUT

SEE ALSO
       docker(1)
